# Workflow Repo

This project is a web application that uses Tailwind CSS for styling, Playwright for testing, and Vitest for unit tests. It also integrates with `live-server` for serving the app locally. The repository is set up with ESLint for code linting, Prettier for code formatting, and Husky for managing git hooks.

## Prerequisites

Before running the project, ensure you have the following installed:

- [Node.js](https://nodejs.org/) (version 14 or higher)
- [npm](https://www.npmjs.com/) (npm comes with Node.js)

## Installation

1. Clone the repository:

   git clone https://github.com/slerbakk/workflow-repo-ca.git
   cd workflow-repo

2. Install Dependencies:

   npm install

3. Running tests:

   npm run test
   npx playwright test
